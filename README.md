# Awesome Design Systems for Paperclip

**58 world-class design systems, packaged as a single [Paperclip](https://github.com/paperclipai/paperclip) skill.**

Give your AI agents access to the design languages of Linear, Stripe, Apple, Vercel, Spotify, Ferrari, and 52 more — so they can build interfaces that actually look like they were designed by humans.

> Based on the incredible [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) collection by [VoltAgent](https://github.com/VoltAgent). All design system files are sourced from their work. This repo repackages them as a Paperclip-compatible skill with routing logic and categorization.

---

## Install

In your Paperclip company dashboard:

**Skills** > **Import** > paste this URL:

```
https://github.com/aaldere1/awesome-design-systems
```

That's it. Every agent in the company can now reference any of the 58 design systems.

## How Agents Use It

The skill auto-routes based on task context. When an agent gets a design task, it picks the right reference:

| Task | Agent Reaches For |
|------|-------------------|
| "Build an analytics dashboard" | Linear, Posthog, Vercel |
| "Design a checkout flow" | Stripe, Airbnb |
| "Create a chat interface" | Claude, Intercom |
| "Make it feel premium" | Apple, Ferrari, Superhuman |
| "Build a documentation site" | Mintlify, Notion |
| "Design like Spotify" | Spotify (exact reference) |

Each reference file includes:
- Typography system (fonts, scale, weights)
- Color palette (with exact hex values)
- Spacing and layout tokens
- Component patterns and interaction guidelines
- Dark/light mode specifications

## What is Paperclip?

[Paperclip](https://github.com/paperclipai/paperclip) is an open-source orchestration platform for AI agent teams. Skills are reusable instruction sets that agents can invoke during their work — think of them as expertise modules.

This skill gives any designer agent in your Paperclip company instant access to 58 production design systems without you having to describe what "good design" looks like.
